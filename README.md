# stalcraft-zone-map-extractor

> Чистая карта локации STALCRAFT / STALZONE прямо из файлов игры — без меток баз,
> точек игроков, иконок квестов и интерфейса. Готовая подложка для интерактивного
> планировщика клановых войн.

Игра хранит миникарту как блочные данные в стиле Minecraft (`.mdat`) в папке
`map_cache/5.0/<локация>/`. Этот инструмент превращает их в изображение:

```
.mdat  --[sc-file]-->  .mca (регион Minecraft 1.12.2)  --[этот инструмент]-->  PNG (вид сверху)
```

Сложную часть — декодирование проприетарного контейнера `.mdat` (zstd-сжатое
квадро-дерево чанков) — делает отличный проект
[**sc-file**](https://github.com/onejeuu/sc-file). Этот инструмент добавляет
недостающее последнее звено, которого не было ни у кого публично: **рендер вида
сверху** (верхний непустой блок в каждом столбце) с натуральной палитрой и
рельефным затенением, авто-разбиение разрозненных закэшированных областей и —
главное — **рентген-режим, показывающий планировку внутри зданий**.

## Примеры

Смотри папку `maps/` — три арены (Хвойный, Малая Бердовка, Низина), каждая в
режимах `roofs` и `xray`. Низина закэширована несколькими разрозненными областями,
поэтому делится на `_part1/2/3` (part1 — полная арена, остальное — дубли-инстансы
и фрагменты).

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

### Без командной строки (GUI)

Для тех, кто не дружит с терминалом — простое окно с кнопками:

```bash
python gui.py
```

Кнопки: «Установить зависимости» (один раз) → «Обзор…» (выбрать папку локации) →
выбрать режим (roofs/xray) и масштаб → «Создать карту» → «Открыть результаты».
Использует только стандартную библиотеку Python (tkinter), ставить ничего не нужно.

### Командная строка

```bash
python scmap.py "<STALCRAFT>/map_cache/5.0/tournament_nizina" -o maps
```

Ключи:
- `-o, --output DIR` — папка вывода (по умолчанию `maps`).
- `-m, --mode {roofs,xray,top}` — режим (см. ниже). По умолчанию `roofs`.
- `-s, --scale N` — пикселей на блок (увеличение nearest-neighbour, зум остаётся чётким).
- `--keep-mca` — сохранить промежуточные `.mca`.
- `--min-area N` — игнорировать закэшированные фрагменты меньше N пикселей.

### Какую папку скармливать

**Любую** папку локации из `map_cache/5.0/`. Инструмент не привязан к трём аренам —
это просто примеры. Что закэшировано в игре, то и отрендерится: можно скормить
`kordon`, `bar`, `zaton`, любой `tournament_*`, `battlefield_*` и т.д. Аргумент —
путь к папке конкретной локации (внутри неё лежат файлы `reg.X.Y.mdat`).

Масштаб — **1 пиксель = 1 блок**, как у самой игры.

### Режимы

В кэше каждое здание хранится столбцом
`маркер → крыша → стены → пол → маркер → комната(воздух) → земля`, где `маркер`
(id блока 190) — **невидимый** слой, который игра никогда не показывает игроку
(он рисует тонкие контур-линии над открытой землёй и накрывает крыши).

- **`roofs`** (по умолчанию) — обычная карта вида сверху: настоящие крыши зданий и
  террейн, невидимый слой-маркер снят (без артефактов-полос).
- **`xray`** — снимает крышу и стены каждого здания до первой комнаты и показывает
  **планировку этажа**. Позволяет увидеть интерьеры/раскладку построек — чего не
  показывает ни внутриигровая, ни любая комьюнити-карта. Прямо для планирования КВ.
- **`top`** — сырой верхний блок, включая невидимый слой-маркер (для отладки;
  показывает те самые скрытые контур-линии).

Файлы получают суффикс режима: `name.png` (roofs), `name_xray.png`, `name_top.png`.

> **Откуда берётся «рентген».** В файле кэша для каждого пикселя карты хранится не
> один цвет, а **весь столбец блоков** (как в Minecraft) — крыша, стены, пол,
> комната, земля. В игре вы видите только **самый верхний** блок. Но если разобрать
> этот столбец и нарисовать не верх, а **пол первой комнаты под крышей**, получается
> забавный эффект: крыши будто спиливаются, и видно планировку зданий изнутри.

## Как это работает

1. **`.mdat` → `.mca`** командой `mapcache` из `sc-file`. Без `--raw` sc-file
   переводит блоки STALCRAFT в ванильные id Minecraft, подобранные под цвет на
   карте, поэтому ванильная палитра близко повторяет внутриигровой вид.
2. Для каждого мирового столбца берётся нужный блок (по режиму) и красится из
   палитры Minecraft 1.12 (`scmap.py: PALETTE`). Высота используется для
   северо-западного рельефного затенения.
3. Разрозненные закэшированные области разделяются проходом связных компонент и
   сохраняются по отдельности.

## Ограничения

- **Рендерится только закэшированное.** Кэш наполняется по мере просмотра/движения
  в игре. Чтобы получить локацию целиком — один раз откройте её карту в игре и
  медленно прокрутите до краёв перед извлечением.
- **Цвета близкие, но не пиксель-в-пиксель.** Они берутся из ванильной палитры
  через ремап блоков sc-file, а не из родных текстур STALCRAFT. Точные цвета
  потребовали бы реестра `id блока → текстура` из клиента (см. «Планы»).
- Декодер `.mdat` в sc-file помечен как экспериментальный — возможны мелкие артефакты.

## Планы

- Точные внутриигровые цвета: сопоставить родные id блоков средним цветам текстур
  (`hrblockMap` / `.ol`).
- Экспорт пирамиды тайлов (уровни зума) + слой точек A/B/C для интерактивного
  фронтенда планировщика.

## Лицензия и условия использования

Инструмент бесплатный и предназначен для **некоммерческого** использования
сообществом. Извлекайте только кэш **своей** игры.

Если вы используете этот инструмент в **платном** сервисе — вы негодяй 😉
(в том смысле, что инструмент создавался как бесплатное благо для игроков, а не
для перепродажи). Делаете бесплатную карту для своего комьюнити — пожалуйста, на здоровье.

STALCRAFT / STALZONE и его ассеты принадлежат EXBO. Это фанатская утилита.

### Про реверс-инжиниринг

Инструмент **только читает ваш локальный кэш** и ничего не меняет в игре, не лезет
в сеть и не даёт преимуществ в бою — он показывает то, что вы и так видите на карте,
просто чище и с возможностью заглянуть в планировку зданий.

## English (short)

Render a clean top-down map of a STALCRAFT/STALZONE location straight from the
game's local minimap cache (`map_cache/5.0/<location>/*.mdat`) — no markers, dots
or UI. Pipeline: `.mdat → .mca` via [sc-file](https://github.com/onejeuu/sc-file),
then this tool renders a top-down image.

```bash
pip install -r requirements.txt
python scmap.py "<STALCRAFT>/map_cache/5.0/tournament_nizina" -o maps
```

Feed **any** `map_cache/5.0/<location>` folder (the three arenas are just examples).
Modes (`-m`): `roofs` (default, clean map), `xray` (peel roofs → interior floor
plans — unique), `top` (raw, shows the game's hidden marker layer). `-s/--scale N`
for crisp nearest-neighbour upscale. Free, **non-commercial** community tool; reads
only your own cache. Credit: `sc-file` by onejeuu.
